Datasets

MicroDraw can be used to annotate any DeepZoom image or image stack available on internet. We are currently hosting DeepZoom versions of several macaque brains from the Allen Institute for Brain Research, and a 1 micron resolution slice of a human brain from the BigBrain project.

MicroDraw uses the Neurolex ontology for cross-species annotation of brain regions and a simple dictionary for annotating histology artefacts.

  • BigBrain human brain slice
  • Allen Institute macaque slices

Application

MicroDraw is based on open source frameworks to ensure its maintainability and extensibility. The interactive multi-resolution visualisation is based on OpenSeadragon. The vectorial annotation is based on Paper. The concurrent versioning is based on Git

MicroDraw is written in JavaScript, using HTML5, CSS3 and jQuery.
